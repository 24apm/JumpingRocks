INSTALLATION

To install this script upload it to your web server and create a database for use with it (it can share a database with other scripts but creates a table called 'contacts'.

You must edit dbinfo.inc.php to contain the details of your database.

Run setup.php to set up the table in the database. You can then delete this script.
